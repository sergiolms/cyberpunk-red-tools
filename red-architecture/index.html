RED RED ARCHITECTURE // Netrunning & Hielo Negro
// CÓMO CONSTRUIR UNA ARQUITECTURA DE RED

Arquitectura de RED

Genera una arquitectura piso a piso: contraseñas, archivos, nodos de control, hielo negro y demonios. Piensa el netrun como un ascensor: cada piso abre una «puerta».

// EL COMBATE EN LA RED

Combate de netrunning

Genera la oposición de un netrun: hielo negro al acecho (con sus stats listas) y, si procede, un Demonio manejando nodos de control.

// PROGRAMAS LETALES

Hielo negro

Consulta las fichas del hielo negro: PER (Deslizamiento), VEL, ATQ, DEF, REZ y su efecto al impactar.

// DEMONIOS Y DEFENSAS

Demonios & defensas

Los Demonios manejan nodos de control para atacar en el espacio de la carne. Aquí tienes sus fichas y las defensas que pueden controlar.
